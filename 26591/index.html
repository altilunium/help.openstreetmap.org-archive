The openstreetmap.de server that creates the GPX tracks is a development server run by the German OSM community. The script that does the GPX exports has been set up by one programmer in his spare time and it is unclear when, or if, it will be available again. I have now "fixed this" by removing the GPX download links from the Wiki.

You can use Waymarked Trails - Hiking

Move the map to the area you are interested in, then click the "Routes" button in the bottom left. Select one of the routes from the list, then it has a link to download a GPX file.
