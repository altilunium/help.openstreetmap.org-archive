Is there a shortcut to select a whole (split) way in JOSM?

1

I find myself being perhaps too conservative when needing to split a way. I get worried someone(maybe even myself) is going to alter the way's tags, but fail to see he/she selected only one part that was split from the original whole way.

Is there a shortcut to select (maybe based on the name tag) a whole way?

I know I can do it by selecting each part of the way while holding Ctrl, but am looking for a shortcut to this.

You could use the search function of JOSM (open e.g. by Ctrl+F): name="some name here" (note that you need quotes if the name includes spaces or some other special characters). Be careful: this simple search term (more complicated ones would be possible to filter more) does not only select streets and does not only select connected ways - so have a look what actually has been selected).
