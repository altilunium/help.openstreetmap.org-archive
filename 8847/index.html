I think these places are are a type of depot, and would be covered by the proposed tag for landuse=depot. Proposed features/Depot.

This tag has been discussed on the mailing lists previously, and is used several hundred times (according to Taginfo)

It would be useful to specify extra details of the depot using the tag depot:type. Possibly something like depot:type=road_maintenance. And you can also tag the name and operator etc.
