It takes a while for new objects to appear on the map. But the shop is there in the data.

The problem in your case is that there are many shops close to each other. Not all of them can be displayed at once. The renderer needs to chose which to display and which to hide. On openstreetmap.org I see the new shop as an icon at zoom level 18. If you don't see it yet try reloading the page in your browser (possibly F5 or Ctrl+F5).
