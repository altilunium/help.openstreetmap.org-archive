I don't think that you have mapped it wrongly - it's a known problem with that map style. See also this issue. Mapnik (the software) is what actually decides where to place icons. There are ways within the stylesheet (or resulting XML) to influence where icons and labels go, but if a change were made it'd need careful testing to see what it broke elsewhere.
