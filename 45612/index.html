Surveyed a part of a way; its highway tag is incorrect; should I fix it?

1

Suppose there is a way of a few kilometers length, marked highway=path. I started traveling along it; discovered that it should have better been marked highway=track instead. Since it didn't match my expectations, I retraced my steps back.

What should I have done in this situation?

  1. Fix the tag to highway=track
  2. Split the way up to the point I have visited; fix the tag for that part to highway=track
  3. Don't bother; leave it as it is

I chose (3) at that time. But what is the right choice? Also, whatever I do, should I add a "fixme" tag describing the situation?

Just to make it clear: this way has no intersections; it connects to other ways only at the beginning and end. So if it looks like a "track" at one end, it's probably the same at the other end.

Just to add a fourth possibility - you've also got the option to ask the original mapper about the "path vs track" question. The easiest way to do this is probably via a comment in the discussion on the original changeset. Obviously it helps to ask a polite question rather than just bluntly say "I think you mapped it wrong", but the only way to find out if there's a valid reason for it being mapped as it is is to ask.
