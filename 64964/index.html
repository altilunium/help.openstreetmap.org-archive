If you search a bit on this website, you might notice that many similar questions have been asked in the past, search for disappear

The reason for the disappearing building

  • not all zoom levels are rendered at the same moment
  • your browser has cached an older version of that zoom level, which did not yet contain the building.

So wait a bit longer, and refresh your browser cache.
