The most easy way if you only want the map on your GPS and not inside any Garmin software on your Mac: do not download the file for "MacOS", but instead osm_generic_gmapsupp.zip. See Garmin.OpenStreetMap.nl:Manual#Installing. Also see OSM_Map_On_Garmin#Installing_the_map_onto_your_GPS (you can usually just rename existing files, while keeping the file name extension).

To install the "MacOS" download you need to use BaseCamp (a Garmin software), which may be the more complicated way.
