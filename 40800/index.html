Este caso não é um problema porquê uma rua pode consistir de várias linhas sim, e os motores de busca levam isso em consideração. Em geral eles só retornam mais de um resultado para uma rua se a classificação de alguma das linhas for diferente (por exemplo, uma linha com a etiqueta highway=tertiary e outra com highway=residential).

Mas se quiser, pode fundir essas linhas sim, já que elas possuem as mesmas etiquetas. Só precisaria adicionar a etiqueta area=no, já que as 4 linhas juntas formam uma área (i.e. um polígono fechado).
