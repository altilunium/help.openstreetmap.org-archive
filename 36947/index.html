When I view 'openstreetmap.org' and click on the 'share' icon at the right side I am by default offered a .png "1:7694" (1307 x 887 pixels) and this once saved and opened in Preview on my iMac displays at pretty much the same size as I viewed the image in Firefox browser. I can also change the "1:7694" to 1:4200 and it will give me an even bigger picture (2395 x 1625 pixels). I can also change the size of the Preview window and/or zoom in to the image to see more detail. If you use windows, the viewer you are using will do the same.

If instead I really do want an image of the about the whole screen instead of just a large image of the map showing on openstreetmap.org, I would use screen capture instead. On the Mac this is done by selecting Cmd+shift+4 and then selecting the amount of the screen I need. This will then give an image the same size as I captured.

...Or are you up to something different?

Maybe you right-clicked the image at 'openstreetmap.org' and selected save image or similar. In this case you will see just an image of ONE of the tiles that make up the view on the screen instead of getting the whole image which is made up of many tiles all joined together.
