If the Bing imagery isn't good enough where you live (and even if it is) you'll be able to contribute much more by collecting GPS data and using that. See the beginner's guide (Nybörjarguide), especially the GPS-data section.

There don't seem to be many Swedish users of this help site, but the Swedish users' forum does seem quite popular.

Och på svensk:

Om Bing bildspråk är inte bra nog där du bor (och även om det är) kommer du att kunna bidra med mycket mer genom att samla in GPS-data och använda det. Se Nybörjarguide, speciellt GPS-data sida.

Det inte tycks finnas många svenska användare av denna hjälpen webbplatsen, men de svenska användarnas forum verkar ganska populärt.
