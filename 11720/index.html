First, it's better to "add new comment" instead adding new answers when you want to answer to another answer. Order of answers is not permanent.

For maposmatic, sorry, it's in the dev version (from last week): http://dev.maposmatic.org/new/

For Bigmap, you can play with the url parameters. You can start from this url: http://openstreetmap.gryph.de/bigmap.cgi?xmin=26&xmax=37&ymin=16&ymax=33&zoom=6&scale=64&baseurl=http%3A%2F%2Fotile4.mqcdn.com%2Ftiles%2F1.0.0%2Fosm%2F!z%2F!x%2F!y.png
