By default Potlatch 2 shows satellite (and aeroplane) photos provided by bing who kindly give permission to derive vector map data for OpenStreetMap.

To enable bing imagery in the merkaartor editor:

  1. Create a new layer via the menu Create → Add new Image layer. If you already have an image layer, skip to the next step.
  2. Change the image layer's source to Bing via the menu Layer → Map → Plugins → Bing.

(That's according to http://wiki.openstreetmap.org/wiki/Bing#Merkaartor)
