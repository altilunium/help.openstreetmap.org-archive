I'd suggest having a look Lambertus' page here. You can choose the country that you want and then select "Download map now!"

As far as "purchase" goes, I'm not sure about that, but you can certainly donate to OSM. I'd include a donate link for Lambertus' page too but there doesn't seem to be one, but he does say:

"OpenStreetMap data and the maps from this site are available for free. But OpenStreetMap totally relies on the input of it's users, so please consider contributing GPS traces or by mapping your surroundings if you find this service useful. Thanks!"
