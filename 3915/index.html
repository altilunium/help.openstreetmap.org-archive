Yes, Merkaartor allows you to add a changeset comment. When you click the upload button, a dialogue box should appear listing the changes you're about to upload, along with an automatically generated comment like "BBOX:-0.62,51.22,-0.58,51.25 ADD:0 UPD:14 DEL:0" -- simply overwrite this text with your comment.
