In OpenStreetMap, you should tag the town according to the guidelines at key:place so that the data is correct. It's mostly about population, but also some other influence of a place's general importance in the region.

As to rendering (how it is displayed on maps), that changes depending on the map. If it is a small town, I would not expect it to display unless zoomed in.

If you're curious, the rendering for the "main" OSM map is on github--I think placenames.mss is the file you'd want to look at.
