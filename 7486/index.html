There are several tags for this, like width= and narrow=yes. Both can be used to specify that a way is narrower than others, but both won't get rendered on the main page. Furthermore there is highway=service with service=alley which might be suitable in some cases, but not in all. Usually highway=service are rendered thinner than other highways, but you should always remember that tagging for the renderer should be avoided.
