The easiest way to get it on the map is probably to add it yourself.

OSM only maps physical presences, so from your website it would appear that that is just the warehouse?

If you allow walk in trade to make purchases then it might be shop=flooring, if not then it is probably just office=company and industrial=warehouse.

A address and contact details would be useful too. Ad copy is not generally considered verifiable information and is usually removed.

Most editing software has a good set of presets so you don't need to add raw tags for most things but most allow you to add custom tags. In the "default" iD editor on OpenStreetMap.org these can be added below the preset on the left panel.

I'd say have a go with the default editor, try to be as precise as possible with the positioning of items. For the most part I think it's fairly straightforward, but learnosm.org give a good step by step introduction if you prefer to learn the software that way.
