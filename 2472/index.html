Yes - ArcReader is part of ArcGIS, but pmf can be only created created by ArcGIS/ArcMAP extension named ArcPublisher (additional license). In ArcMAP you can add OSM as OGC compliant WMS service (http://wiki.openstreetmap.org/wiki/WMS) , and publish it with ArcPublisher.
