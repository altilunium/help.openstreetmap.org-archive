The help text is at https://github.com/systemed/iD/blob/master/dist/locales/en.json (search page for «"help": {» (without French quote marks) or other languages).
Scrolling horizontally currently only works with cursor keys for me. Therefore, you may want to switch (a button on top of the page) the github page to raw-mode text which is easier to scroll in.
However, that all is far less readable compared to the in-editor view (no interpreted markdown syntax). You can paste the code there in dingus, select "preview" mode and hit "convert".

The view in the translation platform (select on the left panel) is more comfortable to read - but still not interpreted (seems to be markdown syntax). (that now requires a log-in)
