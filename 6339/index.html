You should tag them with highway=path and use mtb:scale for bicycle classification. Further tags like sac_scale, surface or trail_visibility might be useful, too. Using these tags allows a better classification than the rather simple highway=trail.

The bicycle tag is for the legal access rights for cycling along that path, on any type of (non-motorised) bike. So this will depend on general access rights in your country, plus any local by-laws or restrictions.

If you have a general right to cycle along that path, it should be tagged as bicycle=yes. If cycling is generally allowed by landowner, that can be tagged as bicycle=permissive. If cycling is forbidden, then bicycle=no.

Then use extra tags (eg mtb:scale) for how easy it is to cycle along it, or whether a mountain bike is required.

This kind of path is often called a "footpad" - a trail that exists only through the action of feet on soil. There isn't a good, official way to describe it in OSM. highway=trail sounds ok, but it has very little usage.

I did just discover "highway=path, path=hiking". Maybe that's suitable?
