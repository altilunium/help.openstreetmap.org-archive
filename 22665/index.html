This looks like a rendering issue. The forest is still there but is now interspersed with areas of scrub and grassland. I think that Mapnik is ignoring the forest and rendering only the sub-areas. To fix the problem you will need to make the forest the outer of a multipolygon with the sub areas inners of the same multipolygon.
