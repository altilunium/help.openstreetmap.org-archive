Maperitive uses a different map style than the map on www.openstreetmap.org; therefore it is possible that while the data is actually downloaded, it is just not shown by Maperitive. You can fix that by modifying the Maperitive rendering rules.

Other explanations are possible but a lot less likely - (1) the data you see in the online map might have been deleted meanwhile, and you're just seeing an older image, or (2) the data you see in the online map might be relatively new, and the database from which Maperitive loads the data might lag behind the master database.
