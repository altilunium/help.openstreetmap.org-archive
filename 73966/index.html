Maybe not the answer you are looking for:

While it is possible to create route relations in iD I strongly advise against doing so. Route mapping is not a beginner task and often requires additional tasks like way splitting, mapping of the bus stops etc. In iD you very quickly lose track of all the relation elements of routes. If you want to create bus routes I recommend familiarizing yourself with the editor JOSM.
