One thing you can do is copy/paste. Select a way, Ctrl + C to copy, Ctrl + V to paste, and you will have a copy of the original way that you can place where you want.

This is different than creating a parallel way. When JOSM and Potlatch2 create a parallel way, it follows the shape of the source way. This allows, for example, creation of co-centric circles, a barrier along a closed way, etc.
