If you want to use OSM date without any guarantee of completeness you could download germany.osm.bz2 and filter for the stuff you want (place=*, addr*, highway=* and maybe boundaries) using osmfilter.

If you really need a complete as possible dataset of german adresses and stuff you'd best have to ask the german government like google did. Though I'd doubt a little the recentness from that source. :)
