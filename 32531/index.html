In short,yes! You need to do a few things. First you create an .osm file. See eg. Download.geofabrik.De for the area you are interested in. Then use a software like mapuploader to create the gmapsupp.IMG file. This you copy into the Nuvi`s map directory a (or use the garmin Mapinstall software to copy this file).
