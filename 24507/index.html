There are very many different maps generated out of OSM data for Garmin devices, some support address lookup some don't (it is a feature that only recently has been reversed engineered). Unluckily the table http://wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin/Download doesn't expicitly list the functionality so you will have to check with whoever produced the map you are using.
