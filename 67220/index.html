The general guidance on the building page of the wiki is that the tag for the building should reflect what it was built as (or appears to have been built as). So if it was built as a fire station (and hasn't been heavily modified) I would tag the building as building=fire_station and place the amenity tags as appropriate on nodes or building:part areas.
