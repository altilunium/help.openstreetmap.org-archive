From what I've seen, people use either English or the language of the region. (In either case, good orthography is helpful so others can use automatic translation tools to decipher it, if needed.)

I use German for edits within Germany (expecting that most people who'd end up looking at the changeset comment for my small-scale edits would be locals), and English in other countries where I don't speak the local language.
