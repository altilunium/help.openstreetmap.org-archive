Wegpunkte oder Abschnitte zu einem bestehenden Track lassen sich folgendermaßen hinzufügen:

Meine Orte -> Tracks -> Track auswählen -> Plus-Button -> Strecke / Routenpunkt / Wegpunkt hinzufügen.
