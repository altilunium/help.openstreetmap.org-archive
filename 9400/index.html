The Project is called "OpenStreetMap", not "OpenStreets".

The map style we offer on www.openstreetmap.org is only one possible representation of the geodata we collect, and is really intended to be more like an "example" than a ready-made map for you to use in a navigation application!

You can download our raw data and use that to make any map you want; depending on the software you use for doing so, rotated maps may or may not be an option. Some vector map clients for mobile devices do offer rotating map display. Check out the list on http://wiki.openstreetmap.org/wiki/Rendering - CartoType is one of the engines that support rotating maps.
