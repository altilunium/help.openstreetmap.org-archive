There is a tomb proposal which suggests to use historic=tomb and tomb=columbarium. One could argue if historic is the right category, but according to the proposal description it doesn't have to be a historic tomb.

You could also use this tag for a dust or scatter field by tagging historic=tomb and tomb=dust_field or tomb=scatter_field. It might be a good idea to mention these two tags on the proposal's discussion page then so that other people know about it.

Also take a look at taginfo to see the currently used values for tomb=.
