If the building is still mostly intact the building tag should normally remain. Any old use that is still visible would normally be tagged either "disused" or "abandoned" depending on how easy it would be to restore it to use (e.g. signs indicate a business that is clearly gone).

The Lifecycle prefix page of the wiki contains better descriptions. Your later examples might be better tagged as as ruins or destroyed. These tags are less common, but not necessarily wrong.
