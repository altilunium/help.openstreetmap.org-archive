OpenStreetMap is a crowd sourced map, so can be mapped what exists in the world. Websites can't be mapped, but offices and warehouses can. You can map your office(s) using the tag office=company. If you don't know how to map it, you can read to the Beginners' guide on OSMWiki.

However, if you need to add a website info to an existing Point of Interest, you must use the tag website.
