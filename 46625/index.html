I checked quickly and afaik there is no specific rule for it and since there is no catchall for amenity (which wouldn't make a lot of sense in any case) it will not be rendered.

You can open an issue here https://github.com/gravitystorm/openstreetmap-carto/issues but I doubt that it will be easy to make a case for rendering it.

You can naturally simply use umap.openstreetmap.fr to create a map that shows the studios, which likely makes more sense in any case.
