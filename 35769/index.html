Have a look at the OSM wiki about android apps. Scroll down this long page until you find a separate table called "Monitoring features" ... and there you see a column about "Open existing track".

About Osmand: I think you cannot select an extra path on SD card only for GPX track files, but you can select the path where Osmand stores all data files including offline maps.

But I think it is easier to copy your GPX files to Osmand's folder by hand in order to display them.
