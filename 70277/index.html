You can try like this:

When you are on www.openstreetmap.org On the right there is option "share" Click on it.

Find place on mape you want, zoom in / zoom out.

On the top you have "include marker" if you need that.

Then there is HTML. Click on it. Copy that HTML code and paste it in wordpress.
