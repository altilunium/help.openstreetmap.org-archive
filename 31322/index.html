Use a TYP Editor to edit TYP file located in folder containing all .imgs. Both TYPviewer and TYPWiz5 allow you to quickly locate the appropriate TYP file.

Restart Basecamp/Mapsource and enter CTRL G to clear previous cache and redraw the map.

There are various tools to replace a TYP file in a gmapsupp, ie Gmaptool
