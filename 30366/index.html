Every OSM editor allows you to add free-form tags that are not in any list. Some OSM editors also allow you to create your own "presets" so that whenever you map a supermarket there will be a field for opening hours.

If you're using iD, don't be misled by the fact that there's an attribute list - that's only intended to help but you can always type a free-text attribute. If you are wondering which tag to use for something, consult the wiki or taginfo.openstreetmap.org where you can search for keywords.
