You (currently) can't. And you should not start using any wild workarounds to change the size of the labels either (e.g. tagging one as an island). That would be considered "tagging for the renderer" which is really frowned upon in the OpenStreetMap community.

There were quite a few discussions about tagging population, relative importance, touristic attractiveness and other features to allow renderers (as well as search engines) to assign priorities to cities (or other items of seemingly equal rank) but so for non of these methods is widely accepted or implemented by the major renderers.
