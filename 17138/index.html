If the Garmin 350 uses the Garmin .img files you can create these with mkgmap from OpenStreetMap data. If you download an area of data, such as from Geofabrik downloads, mkgmap can use it to create the .img files for your Garmin. If the area is too big for a single .img file you can use splitter and then mkgmap to break the map data into chunks. The style of the Garmin map can be changed too.
