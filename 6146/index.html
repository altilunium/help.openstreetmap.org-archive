I presume you're referring to the OpenCycleMap rendering, rather than the standard ('Mapnik') OpenStreetMap data which does not show hills anyway.

OpenCycleMap uses elevation data from NASA's SRTM project. If hills are not showing up, as appears to be the case for Naxos, it is likely a failure of SRTM data rather than anything within OpenStreetMap itself.
