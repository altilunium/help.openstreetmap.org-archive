I think the coordinates in the browser address bar are just the centre of the map view. I don't know if you can set coordinates manually in the default editor.

To edit coordinates directly you will need to use something that gives deeper access to the data. Level0 can do this, but you will have to set the tags manually.

JOSM is a more powerful editor that allows manual input and has preset support so you don't have to enter all the tags manually. Instructions for adding a node by coordinates are here
