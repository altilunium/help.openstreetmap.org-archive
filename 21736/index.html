There is an area with this name: details:

  • name :Fareham
  • ref: Fareham
  • source: naptan_import
  • public_transport: pay_scale_area

A quick search on the wiki gave this page : http://wiki.openstreetmap.org/wiki/Tag:public_transport%3Dpay_scale_area Problems should be reported to "the mailing list". But I have no idea which list is meant.

Maybe it can be discussed on that mailing to change the name to something that includes "pay scale area" ?

There are loads of these Naptan pay-scale areas in OSM. The rendering is a bit silly, in that the name is rendered along the boundary, but the boundary itself is not rendered. They're UK-specific, so perhaps Talk-GB http://lists.openstreetmap.org/listinfo/talk-gb) would be the best mailing list to raise the issue on. Or possibly Talk-transit (http://lists.openstreetmap.org/listinfo/talk-transit).
