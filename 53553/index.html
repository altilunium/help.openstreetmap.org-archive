The "copyright" link at openstreetmap.org should explain: http://www.openstreetmap.org/copyright .

That describes the usage of OpenStreetMap data. If you want to use OpenStreetMap services, you'll also want to read the tile usage policy and the api usage policy. There are also many third-party APIs that use OpenStreetMap data; these will have their own usage policies.
