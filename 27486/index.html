Ne sachant pas quel logiciel/wiki/crm vous utilisez pour diffuser vos articles, il est difficile d'en dire plus. La syntaxe "<slippymap ...=""/>" n'est pas un standard OSM que je connait, peut-être un greffon Wordpress ?

Avec un peu de chance, le greffon que vous utilisez dispose des fonctions nécessaire, allez voir dans la doc correspondante. Au vu de la capture d'écran, je dirais que votre greffon se base sur la librairie OpenLayers. Allez voir de ce coté si vous êtes prêt à améliorer le code du greffon.

Sinon, vous pouvez utiliser OpenLayers ou Leaflet directement (la 2ème étant plus facile d'accès) pour faire ce que vous voulez. Ce sera plus facile à faire pour une page html normale, mais plus compliqué si ça doit être intégré à votre wiki/crm/etc.
