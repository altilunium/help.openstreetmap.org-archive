If you still wanted to use Potlatch 2, then you could draw portions of the border, and add them to a relation for the refuge. Probably it'd make the most sense to make it a type=multipolygon relation, and add relevant tags to the relation. On each way, you'd add them to the relation, and give them a role of outer (or inner).

If the "have an overview to draw the area in the map" is no requirement, yes. But it should be possible to do it with Potlatch2 and one way (no parts and relation):

  1. draw a part of the way,
  2. move the window
  3. continue to draw the way
  4. if the area is not completed yet: go to 2
