The default Standard Tile Layer map using OSMCarto style doesn't render mtb:scale=, or most more technical specific use tags. CyclOSM available on the OSM website does.

https://www.openstreetmap.org/#layers=Y

Refer to https://www.cyclosm.org/legend.html for lines in STS and IMBA scale.
