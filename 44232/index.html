There was an error in the tagging of Shillingford Roundabout: it was mapped as a circular oneway street. I have added junction=roundabout and directions should change to use terms like "take the 2nd exit" (but this will take a while for online routers).

Andy Mackey's suggestion of checking with online routers is always the best starting point to discriminate between software bugs and data bugs. The OSRM instructions immediately sounded very anomalous for traversing a roundabout, so I thought I should check the tags.
