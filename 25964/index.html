It is quite possible that there are small issues with the OSM database server, so your new data cannot be added to the database now. Wait some minutes, and try again.

If there is a bigger problem with some OSM server infrastructure, have a look at the OSM wiki about Platform_Status

Or there are some problems with your internet browser ... close the browser and reopen, or clear browser cache.
