You can set up a local OSM database (see switch2osm.org) and then generate images from that using a program like nik2img or nik4.py (or copy the code from there and add it to your existing backend software). To add a GPX file to the map rendering, one option is to convert that to shape file and modify the map style file to include that shape file. All this, however, isn't available as a free web service - you'll have to install your own server to do it.
