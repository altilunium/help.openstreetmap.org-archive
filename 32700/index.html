There is building:architecture=*, which was part of a now-withdrawn proposal for extending building tagging. Although that proposal was withdrawn, since this particular tag does make sense, and has nearly 5000 uses, that would be my recommendation. Wikipedia has a massive list of different architectural styles you might use for values.
