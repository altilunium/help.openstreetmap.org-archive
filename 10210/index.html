You can always copy OpenLayers.js and the assorted images and CSS files to your local site. This is what many web sites do, including openstreetmap.org

This is however not an OpenLayers support site. Most OpenLayers mailing lists are hosted on osgeo.org so they should still be available even if openlayers.org does not work. See http://trac.osgeo.org/openlayers/.
