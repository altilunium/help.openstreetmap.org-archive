OpenStreetMap is a data project with many applications. It is unclear from your question which application you are using, as openstreetmap.org does not have a "markers" functionality. I assume you are talking about http://umap.openstreetmap.fr . It is possible to import other formats, according to this page (in French). You can also use a csv file. Search this website for "umap csv" to see questions related to that topic.

If you need help with another application, please update your question accordingly.
