You can find JOSM documentation on different locations:
- on the OSM wiki at http://wiki.openstreetmap.org/wiki/Josm (then "JOSM/Guide" -> "Advanced editing" -> "Relations")
- on the JOSM home server at http://josm.openstreetmap.de/wiki/Help (then "Other dialogs" -> "Relation Editor")

The point is that JOSM is continuously evolving by an active group of developpers so the documentation is sometimes a little bit behind the latest features.
