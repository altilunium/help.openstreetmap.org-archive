Is the bus parking a place where bus routes end and people start new journeys? If so it could be a bus station - amenity=bus_station. You can add that in the advanced page of Potlatch. The icons are useful, but other tags can always be added in the advanced page.

Bus parking, would be entered as a normal parking (i.e. amenity=parking), but would required the use of the advanced tagging system to use access tagging (in this case, I believe that is access=no with bus=yes)
