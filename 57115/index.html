The main mission of OpenStreetMap is to create a map. If a building exists on the ground but is missing from the map, feel free to add it to the map right now, even if the satellite photos do not show it.

You can add a building by creating a node tagged "building=yes" and "name=The Building's Name" at the building's location. You can record the name in English as well by tagging "name:en=Building's Name in English" in addition to the local name.

If you have time to take a GPS trace of the building outline, that would be great too, but every contribution is valuable. If you don't have time to take a GPS trace, just add a node so routers (searches) would find the building.
