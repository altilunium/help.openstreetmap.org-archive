  1. Download Osmand's latest Mapcreator from http://osmand.net
  2. Download raw OSM data for the region you need via http://download.geoofabrik.de
  3. Create your own *.obf vector data map file and copy it to your device
  4. Be aware that in certain countries thete have been massive data loss due to the Redaction Bot. Test this for your region first. geofabrik.de is also offering "old" map data before the bot run.
