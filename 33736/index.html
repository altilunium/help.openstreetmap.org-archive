The area is correctly tagged right now as tourism=hotel. There isn't really any need for amenity=hotel, because the tourism tag is the commonly-accepted hotel tag. amenity=hotel has only been used 216 times, compared to tourism=hotel's 148590.

Don't worry about having the area shown in a different colour. The most important thing is to tag the object correctly. The way an object is displayed is up to each renderer and shouldn't affect how you tag something. Don't tag for the renderer.
