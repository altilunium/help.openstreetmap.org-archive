We would like to have the same behaviour as P1, but unfortunately it's not that simple. Unlike ActionScript 1 (as used for P1), Flex makes it very, very difficult to dynamically change the rotation of an icon. It will come in time but it's not something that can be quickly developed.

In the meantime, the next deployment of P2 (maybe this weekend or so) will have an optional 'Enhanced' stylesheet. Choosing this will show direction arrow markers on each way when selected.
