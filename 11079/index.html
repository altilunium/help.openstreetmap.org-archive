A pizzeria can be tagged with the cuisine-key: http://wiki.openstreetmap.org/wiki/Cuisine

amenity=restaurant or amenity=fast_food

cuisine=pizza

The wiki-site also suggest to use oven=wood_fired if it applies (don't know, if this is much used, though).
