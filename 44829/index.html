I've always tended to write the conditional the other way around, so "maxweight:hgv=7.5" and "maxweight:hgv:conditional=none @ destination".

Taginfo stats for the UK are hgv=destination 634, maxweight:hgv:conditional=none @ destination 313, maxweight=7.5 variations around 6000. There are also 428 maxweight:conditional=none @ destination. I suspect a large number of these "non-HGV" restrictions are actually mismapped (I know I'm responsible for a few).
