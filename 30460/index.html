Je vois que tu utilise JOSM. En reprenant en Français les suggestions de cette question:

  • Peux-tu télécharger http://api.openstreetmap.org/api/0.6/relation/161648 avec un navigateur comme firefox ?Si ce n'est pas le cas, il s'agit d'un problème réseau qui n'est pas la faute de JOSM.
  • Vérifie tes paramètres de proxy JOSM en allant dans "preferences -> connection settings -> proxy settings" si tu te connecte à internet via un proxy (copie les parametres de ton navigateur).
  • Ce n'est probablement pas le problème, mais vérifie que tu utilise la dernière version de JOSM (la dernière stable est la 6767).
  • Enfin essaye de remettre à zéro tous tes paramètres JOSM en supprimant le dossier (~/.josm sous Linux/OSX, quelquepart dans %APPDATA% sous Windows).
