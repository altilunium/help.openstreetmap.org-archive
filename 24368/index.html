It sometimes takes a while to render the map once the underlying data has changed, and because it is more resource-intensive, lower zooms are updated slower. There are plenty of questions with answers about this here, if you do a search.

If you zoom in to edit the data, you'll see that the data is correct (the same data is used for all zooms), meaning that the fix has been done already but hadn't been rendered at low zooms yet.
