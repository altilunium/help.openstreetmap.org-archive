Tick 'Layers' under the 'Windows' drop down menu. In the 'Layers' window that is now opened, you can see which layers you have loaded, including the satellite imagery.
You can toggle the visibility by selecting the 'eye' icon, or select the layer and select the trash can or right-click and delete or change the opacity via one of the buttons in the layers window.
Hope this is what you required.
