On the wiki page OSM Map On Garmin/Download you will find a lot of maps for Garmin.

For example this Ethiopia map is updated at least weekly and you don't need email to download it.

With a new Garmin device simply create a folder called "Garmin" on your SD-card and store the .img file in it. Make sure the file is called "gmapsupp.img". For more details on this see OSM Map On Garmin/Mass Storage Mode.
