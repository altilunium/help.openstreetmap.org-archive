As this has not been answered up until now, I will respond with my initial reaction.

Use natural=rock, with an additional tag something like rock=lava. Further keys could be used to add information about the type of lava. Alternatively, if you consider lahar not to be a rock then consider creating a tag value which can encompass other types of destructive debris flows (avalanche, mudflow, landslips, Bergstürzen, and so-on), perhaps natural=debris_flow (my preference) or natural=landslip, again with additional tags to provide more information.

There is also a little-used key geological=*, which may be applicable in this situation.
