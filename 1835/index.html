The versions of the application are slightly different - there are differences in the way the buttons and navigation controls look, for example. But the differences between them are minimal and are mainly configurations differences - different stylesheets, for example, to help fit in better with the rest of the MapQuest Open sites.

The update process is just the same for both of them - the Potlatch2 instance on MapQuest talks directly to the XML API on osm.org, in exactly the same way as the instance on the main osm.org site does. There's nothing to choose between them there.

Feel free to use either copy, or indeed any of the other customised installations of Potlatch2 that are listed on the wiki.
