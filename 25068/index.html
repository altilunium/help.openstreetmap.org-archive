Note that the openstreetmap.org homepage is (currently) not really targeted at end users but more at mappers.

See how-do-i-add-a-marker-to-a-map how to add marker(s) in general (you will see the marker coordinates in the openstreetmap.org URL - mlat and mlong). For routing between positions (markers) see the OpenStreetMap-based routing services (try e.g. http://open.mapquest.com/ ).
