If the edges of sinkhole are vertical then using natural=cliff for it is probably good enough. But if the edges are cone-shaped then natural=cliff is not the tag to use for it. Currently there seems to be no tag used for these natural features. Probably natural=sinkhole would fit. You may also further discuss it on the tagging mailinglist.
