Does this link help?

I have a Nuvi 255W. It takes an SD card (big, old style) so you'll need to get one of those or a suitable microSD adapter, as the internal memory is essentially full. As described by the link above, create a "Garmin" folder on your SD card and copy gmapsupp.img to it.

Then on the Nuvi, Tools / Settings / Map / Map Info, and select the map labelled something like "OSM street map".
