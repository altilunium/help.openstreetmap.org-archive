This is configured in the advanced settings for each layer. By default the "Add permanent label" value is "inherited". This means it is taken from the global settings. Normally the default is no permanent label. Maybe this was set to "yes" ?

The global settings can be found by clicking the "cog-wheel" on the right. For the label they ar under "Default properties".

Those settings can only be changed by the map-maker, not by the user of the map.
