The point of uploading GPS tracks is that you and others can download them and display them overlayed on a base layer (OSM data, OSM renderings or explicitly allowed aerial imagery (e.g. bing's)) in an editor.

I don't know which editor you are using. However, in most OSM editors you can display your own tracks. Some rare editors can only display tracks which are opened from your local computer (uploading still benefits the other OSM mappers) or only uploaded traces. All top three editors iD, Potlatch2 and JOSM can display uploaded tracks and local tracks.

A short how to:

  • iD: For uploaded traces (aggregated(!) from all users): Click on the "background settings" (on the right side) and then enable "OpenStreetMap GPS traces" (note that it may take some time – maybe months – until a newly uploaded trace appears there). For local traces: drag and drop your gpx file (a file symbol on your computer's graphical user interface) to the editor area. See the GPS section of the help inside the editor (bookmark icon on the right side).
  • Potlatch 2: for your uploaded traces use the drop-down menu of the button "GPS" and select "my traces" (or similar). For local traces use Menu "background" → "Vector file" → "Open". See P2 Primer.
  • JOSM: For uploaded traces use the "download data" button (or menu File) (that download all traces in the area, no highlighting of your own, but manual filtering is possible). For local traces use the "open" button or menu file→open. See guide and that other question.

The editor may be the problem. I'm using iD (Potlatch required flash and I haven't been able to get JOSM to load). It looks like OSMtracker is only designed to upload tracks to OSM. So, if I'm understanding you, the correct procedure to use tracks in my case would be this (I've just tried it and it seems to work):

  1. create track using OSMtracker
  2. upload track to OSM with OSMtracker
  3. download track from OSM to local computer
  4. click edit on the OSM track page
  5. click layer icon in editor
  6. click "local gpx file"
  7. open local file dialog
  8. drag and drop gpx file onto editor
