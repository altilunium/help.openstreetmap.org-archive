If you are using Leaflet.js, you could use maxBounds.

Example:

var boundedMap=L.map....
boundedMap.setMaxBounds([[34.161818,-118.591919],[33.683211, -117.734985]]);

to restrict a map.

See: https://leafletjs.com/reference.html#map-setmaxbounds
