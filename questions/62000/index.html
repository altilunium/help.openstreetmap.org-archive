Hi there, it looks good to me. Milano, Roma and the main Italian cities are all mapped the same way

  • Mi https://www.openstreetmap.org/relation/44915
  • Roma https://www.openstreetmap.org/relation/41485
  • Bg https://www.openstreetmap.org/relation/45681
  • To https://www.openstreetmap.org/relation/43992

OSM is a standalone map, it's up to the various software developers using it to find out how to make our data work for them; we don't have to edit things so that they properly show up on third parties software. Feel free to directly get in touch with the Italian community at https://lists.openstreetmap.org/listinfo/talk-it if you want to analyze it further.
