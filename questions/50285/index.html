It may take a day or so disappear. Map layers are not updated immediately. If it does not go soon, paste a link and other mappers can have a look.EDIT I looked at history in that area https://www.openstreetmap.org/changeset/40096199#map=17/43.17396/-73.33627 and the deletion as gone from the data but does show at zoom 14 still,( it is only an hour or so since your work ) The various map tiles are updated so these must be the last to get updated. Ctrl F5 sometimes clears old cached data from some PCs.
