  1. Open josm with at least a bing layer, zoomed at your favorite location
  2. Go to file -> session -> save session as
  3. Tick only the bing layer, and save the *.joz file somewhere
  4. Change your josm shortcut (or however you start josm on your system) to add the *.joz as first argument.
