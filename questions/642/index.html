This is probably not the correct place to report bugs in services. For errors in Nominatim (and most other Open Street Map software) you are best off using the Open Street Map bug tracker which is

To use the bug tracker you will need to log in using your Open Street Map user name and password and then go to "New ticket". Please make sure that you select the correct Component (in this case Nominatim) so that bug report gets processed as quickly as possible.

I've created an example of how you might have presented your bug as ticket #3163 and marked it as fixed since I have now resolved the issue.

The way HaGat does not have any address information. There is no way looking at the OSM data to know where "HaGat 10" is. To find out more about address tags go to the wiki and look at examples in the area. Addresses will show up on mapnik and osmarender.
