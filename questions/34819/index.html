Frederik's answer was no doubt correct last year, but now their FAQ:

http://www.teasi.eu/en/service/faq/

Has a "create your own maps" link, using MOBAC:

http://wiki.osm.org/wiki/MOBAC

which in turn can be problematical because of the load that it puts on OSM's servers, so be sure to follow the tile usage policy:

http://wiki.osm.org/wiki/Tile_usage_policy
