Generally you should file a new issue at https://github.com/openstreetmap/openstreetmap-website/issues .

However, in these two particular issues:

  • Translations are handled at Translatewiki: see https://translatewiki.net/wiki/Translating:OpenStreetMap
  • This is a long-standing issue and one which is unfortunately very complex to fix. It's already known and there's no need to make a duplicate issue. That said, if you'd like to contribute some code, https://github.com/ppawel/openstreetmap-watch-list is the place to start!
