I would draw a polygon of the hard area with the main service roads. Any paths or tracks should connect to the roads for routing purposes.Prices, retrictions,the number of spaces should be added and possibly height restrictions so hiking groups or camper vans may know it's suitable. I think adding all the lanes is a bit excessive but in some areas mappers do the lot and that is ok if consistant with other work in the area. see http://wiki.openstreetmap.org/wiki/Tag:amenity%3Dparking
