The OSM database does not map contours. The contours on the cycle map are generated from the SRTM data set from NASA, which you can download outside OSM.

Note the SRTM data only goes up to 60° north. So it won't cover all of Norway.

There is other height data available, eg ASTER which goes up 83° north. I think OpenCycleMap uses this for some areas.
