Your question is hard to understand.

GPX is a data format, Landsat is a common name for a set of satellite images, and ArcGIS Explorer is a piece of software.

Using Landsat images for drawing things on OSM is allowed since Landsat images are in the Public Domain. Most OSM editors will have an option to show a Landsat imagery background. However this has nothing to do with the GPX data format, and nothing with ArcGIS explorer either!
