Hi, don't worry, different map layers has different update intervals. The cyclemap in northern Italy takes about 14 days to update, while the main layer usually needs few tens of minutes. Remember to refresh your browser page using Ctrl+F5, so that the browser cache gets deleted, otherwise you'll keep seeing the old tiles.
