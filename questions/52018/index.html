You cannot change the symbols that are used on the standard map rendering. That is, unless you install your own version of the server and tweak the rendering (aka drawing) rules.

This is exactly what the French community did, see e.g. http://tile.openstreetmap.fr/?zoom=15&lat=48.85496&lon=2.36157&layers=B0000000FFFFFF for how they render metro stations (already similar to what you want, an "M" in a circle).
