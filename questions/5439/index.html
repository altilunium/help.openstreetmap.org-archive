Well you can do some of those things. e.g. find traces uploaded by a particular user. Here's mine...

http://www.openstreetmap.org/user/Harry%20Wood/traces

At least those are the one which are available on public GPS trace list as per my visibility choices You can navigate to that URL from anyone's user page and then the 'traces' link

"by location" you can kind of do, but not within the trace listings on the website. You can download whole cloud of GPS points at a particular location using one of the editors. That's fetching a bounding box with everyone's GPS traces all mixed up. Some of these points (again as per the Visibility Choices) are associated with the trace upload listing which they're part of.

Users can set tags on their GPS traces. These are different from the tags seen within the main OpenStreetMap data. They're just single words, and probably not very useful for anything really. e.g. here's all the traces tagged "river" ...but that doesn't mean they represent the path of a river. Could be someone out on a walk which was partly along a river.

Think that's everything. We don't currently offer any bulk download options for GPS data.
