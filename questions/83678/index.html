Do not add this golf course to OSM. For the last year or so, OSM has been plagued by people doing exactly what you propose here - making short-term modifications to OSM in order to create an export for a golf course simulator! OSM is not a staging system for golf course simulators. Please contact the makers of the software for a way to achieve the desired goals WITHOUT modifying OSM.

Note that maps and similar works are protected by copyright. Assuming you have permission and the project is at a stage in which is actually going to be built (in general project proposals are not really something that should be added to OSM), the simplest solution using iD is to use https://mapwarper.net/ and add the processed imagery as a custom tile source.
