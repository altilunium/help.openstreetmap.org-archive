You should really check the documentation of each tag on the wiki, but the ideal is that all tags that can be mapped as an area is mapped as an area. In many cases the detail level of a survey doesn't allow for this, in those cases map the node only, or if you have more detailed data than what is already entered, update the tags so that we get the best details possible on our map.
