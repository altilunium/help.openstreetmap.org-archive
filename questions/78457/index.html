THE OpenStreetMap is one big public database. Making any standard edits in one of the common OSM editors will result in them being reflected in that database. You should not add any private or fictitious data there.

Having that said there might be ways to achieve what you want. Unfortunately, it's not very clear what that is. A few options:

You can use uMap to overlay points of interest, ways, areas etc. on top of one of the many OSM based maps.

You could use the editor JOSM to make changes to the map but instead of uploading them to the public server you keep the edits locally and view them in JOSM only.

You could clone the OSM database or part of it (e.g. for a region) and make changes to that clone. Then set up some rendering software to generate maps.
