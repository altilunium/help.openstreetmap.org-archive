You can edit any data you like, but (especially if you're new to mapping) it's a good idea to be careful about what you're doing at first.

If you have any questions, or are unsure about how to do something, there are lots of options that allow you to contact other mappers. IRC can be particularly helpful as it allows you to talk to people in real time.

Also worth mentioning when asking "have I done something correctly" is to include a permalink (using the button at the bottom-right of the map page) to the area that you're working in.
