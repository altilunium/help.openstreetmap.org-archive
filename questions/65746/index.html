OSM is kindly allowed to use bing's aerial imagery to trace from it. See wiki. Bing belongs to Microsoft. There are other permitted imagery providers - just see the selection list in your editor.

I (with uMatrix in Firefox) when using the iD editor on the website do not need to allow JS from that domain - just images if I want to see the aerial imagery (of course). If you struggle with the website-builtin editor, just use another editor (e.g. JOSM).
