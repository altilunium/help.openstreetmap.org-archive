Split it. A way can have a maximum of 2000 nodes. But better don't create a 2000 node way at all to leave some room for further changes. Just create a boundary relation consisting of multiple ways. See Brecon Beacons National Park (loading takes some seconds) for a rather complex example or Binevenagh AONB for a simple one.
