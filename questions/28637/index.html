AFAIK there is no standard in GIS/CAD that says anything about how pan/drag actions should be invoked by which mouse button.
Even in the "expert" preferences dialog you are unable to alter the default behavior, so I recommend to start a feature request ticket: http://josm.openstreetmap.de
