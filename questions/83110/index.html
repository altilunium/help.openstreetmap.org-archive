I've stumbled about the same problem in the past. It says "new note" in the title but in fact only is a new comment. Sometimes you even receive "new note" feeds of old notes without any apparent change.

I don't know of any way to work around this. You can try to leave a feature request on Github to change that in the future. A simple fix would be to have different titles for the messages so one could properly filter in omne's reader.
