Sometimes you can see that the routing engine isn't using up to date data by other means - for example https://www.openstreetmap.org/directions?engine=fossgis_osrm_foot&route=32.67791%2C-16.81438%3B32.67709%2C-16.81326#map=18/32.67774/-16.81320 shows that the road positions that the router is using for the southern road doesn't match the current location in OSM. You moved https://www.openstreetmap.org/node/1970364651 a couple of hours ago and you can see that the router is using the old location.

(these links will of course cease to be useful when the routers have caught up!)
