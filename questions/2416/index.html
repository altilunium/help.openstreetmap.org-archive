You could use snowplowing=no. "Snowplowing" is widely used so far (more than 15000 times!). However, snowplowing=no is only used about 2000 times. See also the wiki: Key:snowplowing

Another possibility is to use winter_service=no for this purpose. According to Taginfo this is used 178 times so far.
