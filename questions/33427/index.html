Hi Grahaml, with P2 just push shift and push the left mouse button and there’s a new node in the way. Add all the tags to the road before cutting it, with the cutting tool the scissor hand in the right lower corner. The tutorial is here http://wiki.openstreetmap.org/wiki/Potlatch_2 Joining them is selecting them with your pointer again while pushing the ctrl button and the chain button in the right lower window. In JOSM it’s done by these rules http://wiki.openstreetmap.org/wiki/JOSM/Plugins/utilsplugin2#Add_nodes_at_intersections_.28Shift.2BI.29 If you’re using ID I don’t know but there is a tutorial as well, http://wiki.openstreetmap.org/wiki/ID
