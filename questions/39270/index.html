This is not the best place for feature requests for the main web site.

Please see https://github.com/openstreetmap/openstreetmap-website for that. However you should be aware that there is only a very small number of actually developing the code. The best way to get results is to set up an instance of the software yourself (see http://wiki.openstreetmap.org/wiki/Develop) and build at least a prototype of the functinality yourself.
