Is that possible at all? Are the map file (ebmd) format specifications available? In general: ask the manufacturer because they openly advertise the OSM-based map function.

I did a bit of searching for you:

On https://wiki.openstreetmap.org/wiki/Medion I see no mention of OSM maps for those devices. However on https://wiki.openstreetmap.org/wiki/DE:Liste_der_Navigationsgeräte a similarly numbered device is listed as with "OSM option" (but not how and where).

It seems you can download OSM-based maps via the manufacturer's (medion) software, see https://wiki.openstreetmap.org/wiki/Hardware_for_OSM-Routing and http://blog.openstreetmap.de/blog/2012/05/medion-s3857-update/

I have found http://www.gopal-navigator.de/downloads.php?do=cat&id=53 (disclaimer: I do not know this site) which claims to offer OSM map downloads for those devices.

Please, mention (in a "comment" or "answer") which method worked for you (for others with the same question).
