I think the question is quite similar to e.g. how-do-i-add-a-marker-to-a-map except that you would have to simply take a screenshot after placing the markers, right? I think the new uMap service (see my answer there) would be suited for you.

If you want to make a "illustration" anyway, exporting (also possible in SVG vector format) and placing the markers yourself in an image editor software would be another option.

If you want to publish something OpenStreetMap-based please do not forget to add the proper attribution.
