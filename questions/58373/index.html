This will work using iD Editor:

  1. Go to your trace
  2. Click the "edit" link
  3. Your GPS track will display in the editor
  4. (iD specific->) Open Background Settings (shortcut B) and select OpenStreetMap (Standard)
  5. (iD specific->) If you want to view with less data in the way, you can also go to Map Data (shortcut F), expand the Map Features section, and uncheck classes of features.
