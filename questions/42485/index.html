("how do I delete this note without resolving it")

There are two ways that notes disappear from the main map - one is 7 days after they've been resolved (though they can still be extracted via the API after that time). The other is if they're hidden by an administrator (which only happens in the case of abusive or spam notes). This one isn't one of those so just resolve it and wait 7 days.
