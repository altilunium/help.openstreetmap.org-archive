The "Add imagery dialogue" is in JOSM. See http://josm.openstreetmap.de/wiki/Help/Preferences/Imagery#AddcustomWMSimagery

You paste the WMS service URL (see above) in the "Service URL" field and click "Get Layers".

The URL you got from the export section of mapwarper is for old versions of JOSM, where it would append the parameters at the end. This might still work, but the only official way in the current version is full URL with substitution markers (like {width} and {bbox}). See http://josm.openstreetmap.de/wiki/Maps#Documentation

Potlatch does not support WMS. Instead, you need to use the "Tiles (Google/OSM scheme)" link provided by MapWarper, and add this to Potlatch 2.

MapWarper will give you a URL like this:

http://mapwarper.net/maps/tile/3/z/x/y.png

You should add the $ symbol before z, x, and y, to make:

http://mapwarper.net/maps/tile/3/$z/$x/$y.png

Then add this to Potlatch 2's background imagery dialogue.

In JOSM, TMS is recommended as well, the format is

http://mapwarper.net/maps/tile/313/{zoom}/{x}/{y}.png

If you like to use WMS, the URL template would be

http://mapwarper.net/maps/wms/313?FORMAT=image/jpeg&VERSION=1.1.1&SERVICE=WMS&REQUEST=GetMap&Layers=image&STYLES=&SRS={proj}&WIDTH={width}&HEIGHT={height}&BBOX={bbox}

This can be retrieved by entering the WMS service URL (http://mapwarper.net/maps/wms/313?) in the "Add Imagery URL" dialog.

One thing to note:

Going straight to the Imagery menu, to attempt: Imagery -> Add Rectified Imagery

is almost certain to create frustration. It is not the same as doing this through: JOSM -> Preferences or File -> Import Image

JOSM -> Preferences is for either WMS or TMS and will add the appropriate underlays to the Imagery drop-down menu. File -> Import Image is for uploading a locally-stored GeoTIFF, etc.

Imagery -> Add Rectified Imagery is for WMS only and is looking for a URL in a format that the JOSM->Preferences approach will create for you.

That all said, TMS will work much better than WMS for zooming, etc.

It's possible to use WMS imagery in Potlatch2 through a proxy service like Whoots.

Use it like this:

http://whoots.mapwarper.net:80/tms/$z/$x/$y/WEB_MERCATOR/http://api.maps.vic.gov.au/geowebcacheWM/service/wms?VERSION=1.1.1&TILED=true

I have verified that this works.
