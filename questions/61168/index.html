See https://wiki.openstreetmap.org/wiki/Tag:shop%3Dclothes ... I guess it is mainly (or at least 33%) clothes? That combination would do:

  • shop=clothes
  • clothes=women;men;children;shoes;hats
  • shoes=yes

The second tag's value is an example, please adjust "women;men;children" to what it really is. Add "leather", too, if it sells leather clothes.

Likely no data consumer will recognize that it also sells shoes... but this tagging is the only way to go since shop=shoes uses the same key. Same for shop=leather (for leather goods).
