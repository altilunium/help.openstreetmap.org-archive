It could be argued that this is one for trac - i.e. that the "GPS Traces" tab shouldn't disappear under the login box (or map). However, it's not too surprising that the problem does occur. If you're using large fonts on a small screen something's got to give, unfortunately. There is code to display "View Edit History Export GPS Traces" in a smaller font if the browser window size is small enough to cause the blurb at the left not to be displayed, but that doesn't kick in for large fonts.

If you reduce the font size for the website (with ctrl-minus) "GPS Traces" will appear again, but obviously the font used for everything else will be smaller too.
