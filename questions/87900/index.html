No, that is not possible. Most maps are generated by producing image files for a given zoom level following fixed rules. You might find some other style (https://wiki.openstreetmap.org/wiki/Using_OpenStreetMap#Ready-made_online_maps) that produces more details for your desired zoom level.

One way that you can sort-of achieve this is by zooming the browser out (usually control mouse wheel down, in a bit of website that is not map) and then zoom the map in.

The text will be smaller, but it will show more detail.
