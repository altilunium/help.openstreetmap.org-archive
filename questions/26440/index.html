In short:

  • Get an account on osm.org
  • log in
  • on the map on osm.org go to the feature which is not correct
  • click on "edit"
  • select the feature by clicking on it
  • change the name (maybe also brand=, ref= or operator=) to the correct value
  • save the edit using a good comment
