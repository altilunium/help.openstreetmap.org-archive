I only have an A4 printer and to see some detail a series of screen prints would be the simplest solution. I have exported some small area pngs and jpegs but I expect that if you try to export large area the request will be rejected because it puts too much load on system resources. This bit in the wiki will give some more info http://wiki.openstreetmap.org/wiki/OSM_on_Paper#As_a_big_image_map
