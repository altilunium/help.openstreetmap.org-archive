Use Natural Earth shapefiles http://www.naturalearthdata.com/downloads/

These can be converted to OSM formats using ogr2ogr or possibly read into JOSM. (Low zoom levels on the main slippy map use this data already).

If you're after simplified country data, a source like Natural Earth is a better option. If you particularly want all the OSM border data and just the border data, the osmborder_filter program from https://github.com/pnorman/osmborder is able to generate this.
