This is not possible from the OSM web site; you can only export those styles which are rendered by OSM itself and that's just the OSM style, all the others come in from third-party servers.

Enter "export" or "render own map" in the search box above to find some information about map export and styling, and also see the answers to "Exporting MapQuest Open Map".
