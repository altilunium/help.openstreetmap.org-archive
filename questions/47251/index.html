Go to the site at http://garmin.openstreetmap.nl and look for the map of Thailand under the Asia pulldown menu. You select the map and provide an email address where you can be notified when the map is ready. The email you receive will have a link to a page from which you can download an installable file that will work in Basecamp. From within Basecamp you can then install it onto your Garmin device.

I've been using these free and fully routable maps for years and they work very well. The installable maps come in several flavors; one for Windows, one for Macs, and a standalone gmappsupp.img file among others. Also, on that main page under "Choose your map type" I advise selecting the "Generic Routable (new style)" map which IMO offers an improved appearance over the "Generic Routable" choice.
