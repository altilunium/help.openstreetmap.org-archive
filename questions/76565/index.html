Have a look at uMap if that suits your needs (e.g. http://umap.openstreetmap.fr/). If you have an own server and want to be more flexible you might want to set up a map with Leaflet.

These two ways work if you are using the map fairly privately. If you want to share the map with a larger audience with many hits you will probably exceed the usage limits of the free maps. In that case you would have to resort to a paid service or set up your own map server.
