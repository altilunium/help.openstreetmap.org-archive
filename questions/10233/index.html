You can add feature requests for Potlatch 2 at http://trac.openstreetmap.org/ , making sure you select the right component.

Your chances of getting the requests done are greatly improved by saying what OSM tags you would like it to have, and by providing icons consistent with the rest of the P2 icons (we use the set from www.sjjb.co.uk/icons).
