As imagens são providenciadas gratuitamente pela Microsoft para uso nos traçados pelos usuários do OSM. Verificando a área, realmente não há nenhuma imagem de qualidade usável lá. A Microsoft (por experiência própria) atualiza algumas áreas mais frequentemente do que outras. Infelizmente, não divulga essa informação, então não há como saber.

Dado que Trajano de Moraes, RJ é uma cidade minúscula (comparado com Rio de Janeiro, por exemplo), é possível/provável que demore muito tempo.

De qualquer forma, se isso serve de consolo, ainda há um enorme Brasil para mapear com imagens de boa qualidade...

(in english)

Microsoft provides, free of charge, its aerial imagery for use for tracing by the OSM's users. Checking the area, truly there isn't any aerial imagery of enough quality for tracing. Based on personal experience, Microsoft updates the aerial imagery of some areas more often than others. However, unfortunately, Microsoft does not usually publishes this information, so there is no way to know.

Since Trajano de Moraes, RJ is a tiny city (compared with Rio de Janeiro, for example), it is possible/probable that it will take a long time until the next update.

Anyway, if it serves as a consolation, there still exists a huge Brazil largely unmapped with good quality aerial imagery...
