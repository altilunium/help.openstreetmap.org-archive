You can use Overpass API instead. Just try instead

http://overpass.osm.rambler.ru/cgi/xapi?node[amenity=hospital][bbox=lon1,lat1,lon2,lat2]

Please note that the order is (on both services) "lon, then lat". For example

http://overpass.osm.rambler.ru/cgi/xapi?node[amenity=hospital][bbox=7.0,50.6,7.3,50.8]

gives all hospitals around my home (Bonn, Germany, roughly 50.6 north, 7.1 east). If you need meta data, use

http://overpass.osm.rambler.ru/cgi/xapi?node[amenity=hospital][bbox=7.0,50.6,7.3,50.8][@meta]

instead.
