Hi.

As you're talking about Enedis, I assume you're asking about France.

There are some datasets from Enedis on data.gouv.fr. Others on their own open data portal.

Some imports to OSM have occurred, but I don't think the data is complete yet.

Enedis also offers a (raster) map of most lines, color coded for capacity, but I don't think you'll be able to use that data for anything else than visualization.
