Tools like http://bigmap.osmz.ru/ do allow you to download a larger image in the style you want. Standard layers on OSM.org are always images. I'm not aware of any service that provides svg's for rendering. You might want to look into rendering vector tiles, see for an introduction: http://www.slideshare.net/OSMFstateofthemap/diy-osm-pbf-vector-tiles-state-of-the-map-2013
