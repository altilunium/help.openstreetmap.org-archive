Das wird erst mit der kommenden Version 1.1 von iD möglich sein. Für Abbiegebeschränkungen sind Relationen notwendig, die bisher nicht von iD unterstützt werden, dafür aber von vielen anderen Editoren wie beispielsweise Potlatch 2 und JOSM. Es existieren sogar eigene Anleitungen zum Eintragen von Abbiegebeschränkungen für Potlatch 2 und JOSM.
