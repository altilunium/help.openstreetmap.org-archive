It covers the area west of the A487, roughly. I cannot see it marked via search, nor can I see it displayed. I could add a marker, somewhere in the middle - but how would one make this sufficiently prominent - say about as prominent as the name for Ynys Môn / Anglesea?

It's already mapped as https://www.openstreetmap.org/relation/9237365 . In general, if you wonder why a particular map does not show that as you would expect, you'll need to ask the map style developers about that. For OSM's "standard" map, that is https://github.com/gravitystorm/openstreetmap-carto .

In this case, it's broken because there was a gap in the outside edge, which you could see in JOSM's validator. I've fixed that in https://www.openstreetmap.org/changeset/98147860 . Various maps will start updating in about 5 minutes.

This isn't a National Park or AONB (well, there is a separate Llyn AONB), but what I wrote at https://www.openstreetmap.org/user/SomeoneElse/diary/395232 mostly applies here too.

Pretty much no, actually - you can only add tags, and then it's up to map styles to decide what to display and when based on explicit and implicit tags and values (an implicit value might be "how big something is").
