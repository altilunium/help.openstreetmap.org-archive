Rule number 1: do not delete things if you don't understand what they are for.

What you are seeing are so called "address interpolations" see for example http://wiki.openstreetmap.org/wiki/Addresses#Using_interpolation They are not a complication, they are a simplification. The alternative is to map individual house numbers, either as nodes, or on building outlines. It is not necessary to map individual lots (but you can do so if you want). If you want to map the individual numbers I think it is a good idea to map the building outlines (makes it easier to associate the numbers with the right buildings). If you do that I would simply add the individual numbers then delete the address interpolation on a block/road by block/road basis.

My personal priority list for mapping

  • major roads (with names)
  • resential, service roads (with names)
  • tracks and paths
  • house numbers (with building outlines where imagery is available)
  • POIs

The nice thing about OSM is that there is no single such list for everybody, so if you don't want to map buildings, just don't, just the same for lots (it is difficult to make a case for mapping them in the first place, but nobody will stop you).
