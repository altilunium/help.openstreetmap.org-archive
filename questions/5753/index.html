When you are brave and get JOSM in use (it is worth it!) then you can even try the "Duplicate" function:

select an existing object from the downloaded OSM data, move the mouse to the target position and simply press Ctrl-"D" as often you need that object.

When necessary you can rotare some shapes later manually.

There is a way, of sorts.

1) Select existing shape

2) press P (or click the 6-spot shape in the tools palette). This creates a "parallel" way, ie a geometrically similar shape (but adjustable size) with the same geometric centre, which will become visible as a thin black line when you move the pointer

3) "wobble" the pointer slightly, firstly to make the new shape visible and then to make it coincide with the old shape

4) click. The cloned shape is now highlighted and can be dragged to the required position.

Obviously it would be more elegant if you could omit step (3) thereby creating a shape which is automatically the right size, but for some reason this creates a shape missing the final edge. I would call that a bug.
