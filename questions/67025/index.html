Thanks for wanting to use OSM! :-)

In case your are talking of the default map you see on https://www.openstreetmap.org the answer is: no, you cannot toggle the labels (e.g. street names).

However, OSM data is free and other map styles using the same OSM data exist. See the older question: https://help.openstreetmap.org/questions/40664/how-to-hide-all-labels-of-osm . You could use the no-labels map by wmflabs.org : https://mc.bbbike.org/mc/?lon=13.368594&lat=52.459748&zoom=10&num=2&mt0=osm-no-labels&mt1=mapnik&marker= However, I could not find out under which license their map is available, so I cannot suggest to use this one.

Since OSM data is free, you can - likely just theoretically for you - make your own maps out of this data.

In any case, please obey https://www.openstreetmap.org/copyright .

If you are planning to print this novel on paper, I would recommend exporting the images as PDFs.

A free option would be to use https://print.get-map.org/new/ which offers several different styles of maps, all based on Openstreetmap data. Perhaps one of the styles there will have the features that you want, but you will need to check that the license matches, if you are planning to publish this novel.

If you want a professional-looking map with just the right labels, you may want to pay for one. There are a number of sites that will help make a custom map: https://wiki.openstreetmap.org/wiki/OSM_on_Paper

For example, https://inkatlas.com offers standard maps for free, but you can pay a few euros for a custom map. There are several other options at the link above.

Alternately you can build a tile server and edit the style sheet and data to show exactly what you want. This can be "free" but may take up quite a bit of your time. If you are using MacOS or Windows you would need to download Docker from https://www.docker.com/, then follow the instructions here: https://github.com/gravitystorm/openstreetmap-carto/blob/master/DOCKER.md

Or if you use a Unix-based operating system you probably will understand these instructions: https://github.com/gravitystorm/openstreetmap-carto/blob/master/INSTALL.md
