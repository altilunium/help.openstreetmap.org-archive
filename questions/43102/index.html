This is an unsupported service written many years ago and hosted on a personal development account on OpenStreetMap's development server. It is not maintained for public use (and really should be taken round the back and put out of its misery).

I'd suggest that you use MapQuest Open's Static Maps API, which is still supported and which, like all the MapQuest Open products, uses OpenStreetMap data: http://developer.mapquest.com/en_GB/web/products/dev-services/static-maps-ws
