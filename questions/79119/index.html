Hi Kephalonomancy. Welcome to OpenStreetMap.

There are no limitations as to what a new user can do or not. It's more a matter of the editor to use for large areas. The in-broswer editor iD is not so suited for that but still it works.

I recommend that you go to the openstreetmap.org website and load the page of the feature you want to edit. You can do so by entering the name "Naval Hospital Guam (Joint Region" into the search box and search for it. Then in the left pain select the object. It will be displayed on the map with orange outline. Scroll in to a maximum on any point on the outline. Now hit the editor button to open iD. The object should be pre-selected and the name displayed. You can easily edit it now.
