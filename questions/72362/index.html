If this is a reasonably common brand you should check that a canonical spelling is entered in https://github.com/osmlab/name-suggestion-index so that we in general use the same spelling (in real life things might not be so simple for example the outlets using different spelling, but that tends to be an exception).

Then to the 2nd aspect, search engines will typically try to normalize both the search input as the user input as far as possible and depending on the search engine will employ fuzzy search algorithms, see for example http://photon.komoot.de/
