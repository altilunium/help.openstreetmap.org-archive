Openstreetmap.in is on GitHub, and the README says the project uses Mapbox tiles that were customized using Mapbox Studio.

You can use these tiles as specified on the README with the following tile url (containing the map ID and access token):

https://{a,b,c}.tiles.mapbox.com/v4/openstreetmap.1b68f018/z/x/y.png?access_token=pk.eyJ1IjoiamluYWxmb2ZsaWEiLCJhIjoiY2psejFtZG8wMWhnMjNwcGFqdTNjaGF2MCJ9.ZQVAZAw8Xtg4H2YSuG4PlA
