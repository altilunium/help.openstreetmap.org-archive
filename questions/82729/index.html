What's "foundation"?

It's a structure. Keep using bridge:support= for =pier or whatever it is. Don't mix up with =pylon for cable towers.

Add eg disused=yes + historic=yes. Don't use https://taginfo.openstreetmap.org/tags/historic:bridge:support=pier since historic:*= is unclear. If it's designated with any status, add the appropriate heritage= + protection_title=.
