Splitting ways that are part of relations/routes is completely OK, and all modern editors will retain the relation/route membership.

If you have more questions about routes etc. I would suggest signing up to the Swiss mailing list http://lists.openstreetmap.ch/ and asking the Swiss community.

As you can see from https://mtb.waymarkedtrails.org/#?map=16!46.9344!6.798 there is actually something weird there, likely the branch shouldn't be in the relation.
