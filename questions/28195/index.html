No, they don't block OSM files. I don't even think that would be possible because they still use the same map format as with previous devices. OSM files work without problems on Garmin etrex 20/30 even with the newest firmware (currently 3.40). For more information see the Garmin and Garmin/eTrex_series wiki pages.
