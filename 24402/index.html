You can not use the map page for navigation but you can run an app like orux maps or http://wiki.openstreetmap.org/wiki/OsmAnd both are, I think, available on google play. They both have the option of off line navigation and map storage. note they will both use OSM map data and, i think, other sources as well as a choice. You may have to run a gps fix app as well you will have to read orux or osmand instructions to check this.http://osmand.net/en/installation-menu.html http://www.oruxmaps.com/index_en.html
