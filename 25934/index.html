FYI, you can also download the data in OSM format in other ways, like via osm.org (Map Layers > Map Data > Retrieve this area from the API), or via XAPI/Overpass (ie - see http://harrywood.co.uk/maps/uixapi/xapi.html). There are ways to convert OSM to SHP, for example with ogr2ogr http://www.gdal.org/ogr2ogr.html I recently used a command like this: ogr2ogr -where "OGR_GEOMETRY='LineString' -lco SHPT=ARC outfile.shp infile.osm. More options here: http://wiki.openstreetmap.org/wiki/Shapefile
