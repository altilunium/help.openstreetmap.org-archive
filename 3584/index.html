Which editor did you use? Do you know about the beginner's guide? Especially the editing part of the tutorial.

Usually you modify the content of the name tag and upload this to openstreetmap. It may take a while until your modification is rendered on the map. You can check if you had success in changing the tag, if the data-view shows the modified value (hit the blue box on the main map and select "data").
