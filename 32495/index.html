This depends on the editor you are using:

  • With JOSM, you can select multiple ways while pressing the shift key. Then you can modify tags for all of them at once.
  • With Potlatch 2, you can select multiple ways by pressing Control (or Command on a Mac); or you can select all ways in an area by holding Control and dragging.

I'm not good enough with the other editors, so it would be good if others could join in.
