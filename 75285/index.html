A lot of addresses aren't yet placed on the map at all, especially in places where there is no open license data source for addresses (e.g., from a local government). If you have personal knowledge of an address, then it's pretty easy to edit the map to add it yourself: https://wiki.openstreetmap.org/wiki/Addresses

Note though that if the only way you know the address is from those other maps you checked, then that's not information we can add to OSM legally.
