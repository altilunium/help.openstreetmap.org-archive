historic=wreck is visible on http://gk.historic.place, a map showing all "historic=" objects. Requests for changes in the rendering of the "standard" map should be made on https://github.com/gravitystorm/openstreetmap-carto/issues, but it is possible that the maintainers of the style think historic=wreck does not belong in the standard style.
