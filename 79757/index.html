That is not the "secret editing link". Please check your browser history if you can still find the correct one. It should have a string of letters/numbers in the end (not the coordinates as in your one). If you have changed devices or browser go back to the one you originally created the map with. Your access might still be saved in a cookie there.

If that fails you can export the data of your map (share icon - download data), create a new map and import the data there. Better first create an account and log in before creating the new map. Then you will always have editing rights without need for a editing link.
