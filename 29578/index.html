I think you should better ask this at the company which made this app: http://www.locusmap.eu/support/ .

If the app directly accesses the maps (online), then you could check at the homepages of each map to see the update cycle (just search the web for the exact name, note the spelling, or look at this list). You can also look at these map homepages if the update has arrived there already (note that you may need to clear your browser's cache if you have been there before - Ctrl+F5 helps). Note that locus may have a map tile cache (stored on your device) which you may need to empty manually to get a map update.

Just for clarification: OpenStreetMap itself only collects/manages/offers the raw geodata (and one map based on it). Many other maps and services are based on it (e.g. several bike or hiking maps) which are run by independent operators.
