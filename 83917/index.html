The map that you show is a Microsoft map. It claims to contain data from three sources - TomTom, Microsoft and OpenStreetMap. The styling of that map and what data from those sources is actually shown is entirely determined by Microsoft/Bing.

You will need to contact Microsoft about the errors that their map contains.

Separately to this you may want to check that OpenStreetMap in your area is up to date. This won't help fix your Microsoft map, but it will help people who prefer more accurate maps, such as those mostly based on up to date OpenStreetMap data.
