This problem is most probably due to both maps having the same product ID or FID. You will need to use a special software tool to change this before installing the second map. I use JaVaWa GMTK.

This issue is well documented in the main OSM Wiki and can be found using this search. Many providers of OSM Maps for Garmin also have info on their sites. (eg http://garmin.openstreetmap.nl/ FAQ)
