As we see here is seems that the island was deleted (apparently).

I guess it was accidentally deleted when the user tried to delete a newly added supermarket which was accidentally added into the coastline (or even fully accidentally added). More guessing: Being slight off the node when selecting likely caused the coastline to be selected and then the delete command was issued without looking properly which object got selected.

I have undeleted the island's coastline. It is now existing again (with the same id as before). I did not undelete the supermarket (its old position).

Since costlines are treated specially in the rendering process, it may take longer to show up again (a system admin says: nearly daily). By the way: it still is there on some other layers (some apparently had not updated their coastline yet).
