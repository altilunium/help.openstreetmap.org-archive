You can edit the map yourself. Set up an account/ID on the main OSM website, then zoom in on the area you need to correct and under the "Edit" pull down at the top of the page select iD (the newer web based editor) or Potlatch 2 (the older Flash based web editor).

Or, I guess, you could give the latitude/longitude of the area and post the name of the road and somebody could add it for you. It would be better if you added it yourself though as you will like start to notice other errors and omissions on the map and want to correct them too.
