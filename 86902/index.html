No. When you edit OpenStreetMap, your changes are pushed to the official database immediately. Creating a fictional city will get you banned quickly.

You might be interested in checking out this:

  • OpenGeofiction is a sandbox where you can create fictional map data in a separate database.
  • uMap lets you create a personal layer with all kinds of points and lines, and this information does not interfere with the public database.
