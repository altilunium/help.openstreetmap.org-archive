Regarding what I found about Lacul Dracului this is a natural water in a collapsed cave. If you don't want to use
natural=water because it is a pool of a stream/river, you can use
waterway=riverbank to map that pool. Since it is inside a collapsed cave you could also add
embankment=yes and - if matching
natural=cliff (maybe only partly).

After all don't forget: mapping is about simplifying. :)
