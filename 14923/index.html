You need a map display program that uses mapa data that you can generate on your own.

And that runs on both, Android and Desktop PC

The only program that can do that is (as far as I know) Navit

If you need a map with only railways, get an extract of your area from download.geofabrik.de and do a filtering with osmfilter or osmosis.

Consult the OSM-wiki how to use them.

So you know where your railway track is, and it is not included in the map data on the main OSM server (that means you cannot see it on osm.org)?

Then you can add that track to the general OSM data by the use of an OSM editor. See http://wiki.openstreetmap.org/wiki/Beginners%27_guide and read and try, bit by bit.

Or if you don't want to upload your track to the OSM server, create it with one of the offline OSM editors like JOSM or Merkaartor and save the data as an .osm file only to your harddisk. Then feed Navit's mapcreator (or try Osmand and its mapcreator) with your local .osm file.
