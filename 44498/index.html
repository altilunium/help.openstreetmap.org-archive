I've used these settings (coords changed to yours):

    var southwest = L.latLng(51.490, -0.122),
        northeast = L.latLng(51.510, -0.028),
        bounds = L.latLngBounds(southwest, northeast);

    var map = L.map('map', {
        zoom: 8,
        maxZoom: 18,
        maxBounds: bounds
    });

    map.fitBounds(bounds);
