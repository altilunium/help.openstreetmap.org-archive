I don't think you can delete things with MAPS.ME but if you zoom in to the place on https://openstreetmap.org and click "edit" you'll be able to remove it there. If that doesn't work you can either post the URL here or (if you don't want to make it public) email data@osmfoundation.org about the problem and we'll be able to help.
