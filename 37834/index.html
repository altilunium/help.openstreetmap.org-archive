Não tem fórmula mágica ou regra que serve para todos os casos, mas neste exemplo que você deu a forma mais prática, funcional e simples seria:

Esse ponto azul com um triângulo branco é a representação de um traffic_calming=island, que é uma separação temporária da via: http://wiki.openstreetmap.org/wiki/Tag:traffic_calming%3Disland

Só precisa utilizar um ponto representando isso, sem separar as vias. Apenas para comparação com os dados atuais do seu exemplo, caso alguém modifique os dados posteriormente:
