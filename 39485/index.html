Please see the small tooltip in the user field: "comma separated, use !/- to invert".

To exclude your user's changes, simply enter !cb650cx in the user field

I tried this on simon04's version on http://simon04.dev.openstreetmap.org/whodidit/ but it also seems to work on http://zverik.osm.rambler.ru/whodidit/
