OsmAnd 2.5.x

click on "Globe" in left upper corner, Click "GPX-track...", there is an option, "Show current GPX-track" (I translated this from the Dutch UI, so it might be named a bit differently).

In version 3.5.5 it's now found under Globe in left upper corner - Configure map - GPX files - Show GPX data - Show current track (toggle it on) You can configure it differently for your mode: car hike bike etc.
