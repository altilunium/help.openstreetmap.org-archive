A spammer has brought this question to the top of the list again, so I might as well add what is now (9 years later!) the correct answer.

Diary entries have a "report this entry" link below them. Admins will see those messages, and can delete the diary entries and take appropriate action against the user.

In addition, users now have a "report this user" link. This can be used to indicate to the admins that (for example) a user's profile has spam in it.
