Try this http://www.nearby.org.uk/conversions.cgi for the conversion, put the lat lon into osm search box, better if you zoom in first, share and add a marker and you get this https://www.openstreetmap.org/?mlat=53.08871&mlon=-3.37557#map=18/53.08871/-3.37557

Put the co-ords into the search box A. Zoom in by clicking on B + . "Go" on search ( if you don't do zoom first you wont know where the middle is. Use C to share.Add a marker tic box D. Click on E. Then cut and paste link at F. that's how i generated the map with the marker. This one https://www.openstreetmap.org/?mlat=53.08871&mlon=-3.37557#map=18/53.08871/-3.37557

I sounds much more complicated than it is. Easy to try out and not a problem if it goes wrong, I email links to myself to test them.
