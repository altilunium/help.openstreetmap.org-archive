We only use GB OS Open Data, released under the Open Government Licence. We attribute OS open Data in our copyright page as required. We cannot use regular OS data as any kind of source, nor can we use data derived from OS sources such as the recently released UK Land Registry cadastral data, even though LR would like to release it as Open Data.
