The only issue here is: do the curves just have the "curve" name or should both the curve and the street names be tagged on them.

In the first scenario you can simply split the road before and after the curve and set the name to the "curve name" (I suspect google simply does that).

If the latter is true there is currently not a "standard" way to do it, but you could use alt_name for one of the two names.
