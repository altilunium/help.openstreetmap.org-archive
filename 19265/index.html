Why request? OSM is a do-ocracy. If you want something, do something:

  • The TIGER data is in public domain so you can fetch it.
  • There are the import guidelines. (Take care not to skip some for convenience, especially do talk to the locals in whose regions you want to import to!)
  • Put some real effort in all of it, not at last in converting and cleaning the data

-> you can import the data you need yourself.
