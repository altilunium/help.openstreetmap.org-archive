For a web based slippy map like the one that you see at OpenStreetMap.org, the instructions are essentially:

  1. With server online, follow these instructions.
  2. Take server offline.

I'm not sure about "use it in your software" though. What is your software expecting? Does it just want a background map, or does it want to use the OSM data for something else too? With regard to "licence price" see https://www.openstreetmap.org/copyright .
