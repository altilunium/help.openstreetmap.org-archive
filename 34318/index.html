When you are logged in here on this help site, there should be* a check mark/tick symbol () just below the answer up/down vote buttons (, ) left to each answer. Looks like this:

Click that check mark ( → ), and you have accepted this answer and set your question to "answered".

* This does not (yet) work for own answers.
