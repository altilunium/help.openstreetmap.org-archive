As far as I can tell the Garmin GPSMAP 62 does not use prerendered tiles (which would be problematic to download for all of Ecuador) but uses map files in the Garmin .img format. Have a look at wiki.openstreetmap.org/wiki/OSM_Map_On_Garmin for documentation of how to get such a file based on Openstreetmap data.
