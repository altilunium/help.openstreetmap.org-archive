I think the best way to tag it would be amenity=social_facility . Add social_facility=group_home to signify that it is for some special group and is residential and add social_facility:for=drug_addicted to tell the place is a drug user support group.

Reference:

  • http://wiki.openstreetmap.org/wiki/Key:social_facility
  • http://wiki.openstreetmap.org/wiki/Tag:social_facility%3Dgroup_home
