There are many ways to do this with varying degrees of customisation. While OpenStreetMap.org is the source for the data, the services using that data are normally run separately.

This page on the wiki has a list of tools to get you started, or you might be interested in some of the commercial services that do a lot of the server side work for you.

I think it is worth mentioning that the default style you see at OpenStreetMap.org is subject to a tile usage policy. It is mainly there to help contributors and generating a lot of traffic will probably get you blocked. The bottom of that page also has a short list of some of the larger providers.
