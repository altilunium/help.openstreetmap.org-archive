This isn't really an OpenStreetMap question at all.

You need geometries of Census Tracts which are not present in OSM. A suitable place to find them is here.

I would recommend using QGIS which can read both the geometries and your csv file and plot the relevant values. You will find extensive information about this software on the web, and specific guidance about particular problems on GIS Stack Exchange.

The latter would have been a more appropriate place to ask the question in the first instance anyway, and this answer belongs there too!
