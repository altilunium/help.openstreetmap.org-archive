If you found a location, where the name is missing, e.g. a street without a name in OpenStreetMap that does have a name on the ground (that is, if you go there, you'll see a street sign for example), you can start here: https://www.openstreetmap.org/fixthemap by either signing up and fixing it yourself or by adding a note for others to fix it, but the latter option may take some time. It would also be helpful to link to the location here and tell us the name that is missing.

If you are just want a map where that missing name is displayed but that name is not feasible to be included in the OpenStreetMap data set, you could set up a map via a service like umap.openstreetmap.fr and add the name as a marker.
