You have installed the "separate data store" plugin for JOSM which is used by HOT to store data onto a different server in addition to OSM uploading. If you're not involved with HOT, and haven't set up your own SDS server, then you are unlikely to need the SDS plugin. Remove it from JOSM and all will be fine.
