When something new is added to the map data it will always take a while for that data to be rendered by the various maps. If you look at the cycle map it has not been rendered yet. You just have to wait for a little time, rendering used to be slower in the past. Note:- It is much easier for us to look at point on the map if questioner just cuts and pastes the URL, browse map and zoom in then just copy the url as in my comment. If I experience a delay in rendering i just open the editor and check tags and mapping were saved. Sometimes old data may be stored in the browser cache which can cause older data to be displayed.
