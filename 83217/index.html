Hi Wolberdinger and welcome to OSM.

Different maps update with different speeds. The standard layer on www.openstreetmap.org usually updates instantly, at least on zoomed in levels. Have you tried that one? If this does not happen for you you might have to refresh the map bypassing the cache. Often that works with ctrl+F5. In rare situations the rendering server is overloaded so that updates take some more time to appear.
