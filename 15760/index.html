How to set speed limit in mph rather than kph?

2

New editor here: a section of road near me (A24 south of Dorking) has had its speed limit dropped from 60mph to 50mph. I have no problem (in Potlatch2) in selecting the speed limit currently recorded ready to change it. However, all the options for new speed limit to select are in kph, even though the original speed limit is correctly recorded in mph.

How can I tell it to set the new speed limit to 50mph rather than 50kph (or do I have to select 80kph which would be about right)?

You should be able to scroll up to the mph speeds which appear before the km/h speeds in the dropdown list of what's available.

If that fails, select the road, click advanced, and just modify the 6 to a 5 in the "60 mph" bit.
