It is not a matter of a donation or similar. Most of the time, it is just a question of you seeing old data in your browser due to the map tiles being cached.

So that we can debug the issue please supply a link to the object in question: zoom in to the area where your shop is, select "Browse Map Data" from the "Edit" menu. On the left hand side you should see a list with all the objects in view, select the node for your shop and then "Details", and the resulting link to your question.

As EdLoach points out the issue you are seeing is likely due to shop=gift not being rendered (drawn on a map) in the standard style map.

See http://wiki.openstreetmap.org/wiki/Map_Features#Shop for a list of shops with indication what is currently displayed.

Assuming this node is the shop you added, it is currently rendering as just the house number, as gift shops don't have any special rules for rendering in the default map you see at www.osm.org. You can however use the search box on the left of the screen to enter the shop name which will find it.
