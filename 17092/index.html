Die Straße sollte in zwei oder mehrere Teilstücke aufgeteilt werden. Verschiedene Tags (z.B. Einbahnstraße, Höchstgeschwindigkeit) können dann für jenen Abschnitt getrennt angegeben werden. Um eine Straße im Editor Potlatch 2 teilen, wähle den (falls nötig erst erstellen) Knoten, wo die Teilung erforderlich ist, und klicke dann auf das "Schere"-Symbol unten rechts.

Englischer Originaltext: Please excuse the reply in English. The road should be split into two or more sections. Different tags (e.g. one-way, speed limit) can then be applied to each section. To split a road in Potlatch 2, select (or create) the node where the split is required, and then click the 'scissors' icon.
