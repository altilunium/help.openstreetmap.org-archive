I'd only tag private inside parking as an entrance. Indeed there is a not entirely insignificant number of entrance=garage tags.

If you mean the parking is outside, I'd simply put an access tage on it (likely access=private). Connected buildings can be either retraced as separate linked buildings, or as a single building=terrace, if you like that better.
