According to http://wiki.openstreetmap.org/wiki/Key:highway, squares should be specified as highway=pedestrian and area=yes. The additional place=square is apparently more recent and does not suffice on its own.

When you select square on the ui (at least in the iD editor), it only sets place=square, so you must go add the other two as well.
