Hi, This page gives a good picture of building levels tagging:- https://wiki.openstreetmap.org/wiki/Key:building:levels From it you should see a bungalow with a pitched roof would be tagged building:levels=1 (the ground floor), roof:levels=1 (the roof).

With regard to roof levels please see this page:- https://wiki.openstreetmap.org/wiki/Key:roof:levels If an attic/roof serves as an accommodation level it's still tagged as a roof level.
