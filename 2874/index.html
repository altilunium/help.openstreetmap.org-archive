Many different people make Garmin map images available from OpenStreetMap data, using different methods and tools. The most common tool is called mkgmap and currently it has very limited support for creating map images with address search support. Work is underway at present to improve this.

You may be able to find maps made using other tools, or you might have to wait until newer versions of mkgmap support the search fully and people use this future version to create their map images.
