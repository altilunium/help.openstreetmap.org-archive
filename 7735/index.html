P2 won't run offline, unless you have an OSM server set up on your own machine - which definitely doesn't count as "easily"!

However, you can use it on other databases other than the main OSM one. In particular, you might like to take a look at the test intallation at http://random.dev.openstreetmap.org/potlatch2/potlatch2.html . This is connected to a testing database (that at api06.dev.openstreetmap.org), and so you can happily play and save whatever you want without damaging data.
