If you had had a look at the Alps you would have found that they are already mapped – with 

natural=mountain_range

which has 93 occurrences so far and gets used by Nominatim.

See https://wiki.openstreetmap.org/wiki/File:Maxbe-stubaier-beschriftung_en.png for a suggested tagging (place=region + region:type=mountain_area) for multipolygons and possible use in a map rendering. We currently have 215 of region:type=mountain_area in the data, according to taginfo.

Possibly you might be interested in https://help.openstreetmap.org/questions/50096/tag-naturalupland-as-way-or-enabling-way-for-place-tags too)
