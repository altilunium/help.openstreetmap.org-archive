Mapillary (Provided by facebook)
OSC-OpenStreetCam (Provided by Grab)

These are used to add street level photos. You can create an account and add photos. These photos can be viewed in the ID editor and JOSM.
In ID editor:
Path:-
OSM ID editor login -> Map Data -> Photo Overlays -> Mapillary / OpenStreetCam (enable the feature)
