What are you actually trying to do?

Microsoft Visual Studio 2019 is a code editor. It understands projects in all sorts of different computer languages. A .osm.pbf file is a data file. Microsoft Visual Studio 2019 doesn't understand that data file format, so it won't be able to make any sense of it.
