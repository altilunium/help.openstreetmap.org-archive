It does exist, please take a look at the amenity=embassy documentation. It mentions an diplomatic tag to specify whether this is an embassy, an ambassador’s residence, a consulate etc. See current values in taginfo.

In your case please use amenity=embassy with the additional tag diplomatic=consulate.
