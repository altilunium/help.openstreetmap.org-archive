You should put the other tags for a building on that object which represents the "building" - the object which has the buidling=* tag. This the building outline.

The building relations do not have the main tag building=*. "Relations of type=building are used to group elements that form a building in simple 3D modelling." ( https://wiki.openstreetmap.org/wiki/Relation:building )
