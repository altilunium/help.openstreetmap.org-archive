I'm guessing by "transport map" you're referring to this tile layer that appears at the OpenStreetMap.org website. If you want to use those tiles, see the links on that page for how to do so.

It isn't possible to choose "only some features" from an existing layer of raster tiles (though if you had access to the source of the map style you could create your own tiles to show what you want). Other map styles exist that might be closer to what you want - see here for more information.
