dgarner, OSM is a large database that could handle your question if all the data that you want to use is present. If the data is not yet present, please feel free to add it and use the database and search or make a program to draw your plan. The terms of use of OSM can be found here, http://wiki.openstreetmap.org/wiki/Main_Page Here’s a link to a well numbered area, http://www.openstreetmap.org/#map=17/52.32079/4.97408, but remember this building has 11 levels and the numbers are neither right on spot nor level. Or this one with 14 levels with the same manco, http://www.openstreetmap.org/#map=19/52.08288/5.17556 You could use the normal rendering of OSM if the house numbers are present and use an old fashioned printed map.
