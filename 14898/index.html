You have to explain first that the "search bar" is using a tool called "Nominatim". And second, this tool is not searching directly into the OSM database but is using its own database normally synchromized with the main database (and customized for its needs). Last, the Nominatim database is outdated because the synchronisation process with the OSM database is suspended since one month.
