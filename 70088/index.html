Some tag do not render on the default map. The example wells from the wiki don't show on the default style so your well will not show either. The humanitarian layer does show wells with a bucket icon. If it shows up on the humanitarian layer there soon, you have probably added it correctly.

The map style is open source, if you think this should render you can raise a feature request on its github repository.
