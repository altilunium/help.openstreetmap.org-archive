OpenStreetMap cannot control how Strava are using the database.

It seems Strava use the OpenStreetMap database to display the local names on the maps they generate. This is the most common use of OpenStreetMap because it is easiest, and most likely that a name always exists.

Some other users of OpenStreetMap data display specific-language names rather than the local one, but an English option is not currently available for the world. It is also possible to create your own maps, if you have enough technical knowledge and server resources.
