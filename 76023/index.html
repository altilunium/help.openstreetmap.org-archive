With any map that uses OSM for some of their data, you'll need to contact them to ask when the data will appear.

Some OSM-based maps use OSM for some things but not others (Facebook is an example of that) and some might take quite a long time to update after new OSM data is available (that used to be the case with Mapbox, not sure if it ia now).
