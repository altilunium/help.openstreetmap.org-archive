Unfortunately, Creative Commons Attribution 4.0 International License is not enough for import in OSM, mainly because we can't enforce attribution, if I understand correctly.

You have to request a specific authorization from the data publisher.

The list of already accepted requests is on the wiki, there is a specific LINZ page.

Before any large-scale import, please be aware of the import guidelines, it's not a task to be taken lightly.

You should first synchronize with the NZ community, for example through this page.

Good luck.
