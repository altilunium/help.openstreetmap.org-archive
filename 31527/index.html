It takes a while for a change to be rendered on a map and each zoom level is basically a new map rendering. And different map sites that use OSM data have different update periods.

So if the change showed up an any zoom level, you've successfully made the update. Congratulations! Now it is simply a matter of waiting until all the various renditions of the data are updated.
