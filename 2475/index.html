Potlatch 2 on openstreetmap.org doesn't currently have a built-in way of showing unnamed streets, but there are a number of steps you could take to get one:

  1. Create a custom stylesheet for Potlatch 2 which highlights streets with no name. Potlatch 2 uses a language called MapCSS, so you can create any style you want. There's a guide to deploying a customised version of Potlatch 2 on the wiki.
  2. You could ask the developers working on Potlatch 2 to create such a style sheet and add it to Potlatch 2 running on openstreetmap.org. This will almost certainly take longer than doing it yourself, since the devs may have other priorities they want to work on. File a ticket in Trac with the component Potlatch2 and Type enhancement.
  3. You could use CloudMade's NoName rendering as a custom tiled background in PL2. This is only updated once a week, so any names you add won't show up immediately, but it still shows areas that need mapping.

Please bear in mind that for most places, the correct way of getting missing street names is to gather them in a survey. Methodically copying names from another source without permission is a violation of copyright, so please don't just take the names from another map.
