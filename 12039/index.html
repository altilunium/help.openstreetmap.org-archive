The coastline polygons are too complex and errors so devestating to be generated every time the server needs to render a single tile. Therefore the coastline is processed about every two weaks in a costly process that tryes to eliminate a lot of errors and save the server for a lot of processing time during regular operation.

When the coastline shapefile was generated (31 Mar 2012) the way was not a coastline and therefore was not included. Next time the shapefile is generated it will include the island.
