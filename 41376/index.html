Hi Jacob, if you mean just to add one shop/company location, your own, to the OSM database (and, by this, to OSM-based maps)...

Please just add it yourself (please only objective data, no advert texts). You have already registered an account. See e.g. how-do-i-get-started-with-openstreetmap. Welcome to OSM! Of course, we hope that you could add a bit more useful data after you've added your business, but that is up to you of course.

Another option is to add a request for inclusion via a Note or indirectly via http://onosm.org/ (more structured).
