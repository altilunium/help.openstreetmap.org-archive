You may need the Garmin.img to be a of a different type for use a a Garmin GPS. You could download a Swiss version of this http://garmin.opentopomap.org/ which are very user friendly. Once installed on the GPS plug-in the USB lead and Basecamp will probably display it. If you wanted it to be always available to Basecamp put a copy of that same file onto a memory stick and that works for me.
