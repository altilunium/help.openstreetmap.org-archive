Some local communities run their own taginfo instances (e.g., France, UK & Ireland), for a full list see here. Unfortunately there is not one for Spain nor Italy (anymore).

Taginfo does not provide the type of geographical breakdown capability you wish for so overpass may be the best approach. For large numbers of queries perhaps consider Osmium tool and scripts.
