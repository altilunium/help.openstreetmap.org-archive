Hi, you’ll have to find the admin border of Oatley & Mortdale on the map first. Make sure that you have the official area code number, try more than one source to verify it. The Australian admin borders seem to be imported into OSM, so control and ask locals too. Select the admin border. Control the border to see if it’s a closed way. Select the edit mode of your editor, change the code in the advanced menu and save it.
