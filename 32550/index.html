found the problem.

I had the wrong input path in /etc/rendered.conf 

...
[mapnik]
#plugins_dir=/usr/lib/mapnik/input       # for mapnik_2.0
plugins_dir=/usr/local/lib/mapnik/input  # >=  mapnik_2.1 (has not kismet.input)
